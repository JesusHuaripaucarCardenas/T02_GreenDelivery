# GreenDelivery 
GreenDeliveryes una startup que ofrece servicios de entrega ecológica 
en áreas urbanas. El reto fue diseñar una base de datos que permita gestionar
clientes, pedidos, rutas de entrega y conductores, optimizando la asignación
de recursos en tiempo real”.

### Problema: 

* Evitar redundancias 
* Permitir reportes útiles.

### Requisitos clave: 
* Registrar clientes
* Pedidos 
* Asignar conductores
* Rastrear rutas
